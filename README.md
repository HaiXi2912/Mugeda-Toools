# 木疙瘩工具箱
提供更实用的导入功能和快捷操作

目前暂未于Github发布会在4月中旬发布
您可以点一个Star关注后续更新
（后续会更新行为编辑器的接管，让行为好操作）


图片导入处理：
拖拽图片页面后，可选择：
智能导入
裁剪后导入
修改尺寸后导入
裁剪并修改尺寸
GIF压缩后导入

支持将 SVG/WEBP 等转为 PNG 后再参与裁剪/压缩与导入。


音视频：
拖拽音频后先做校验与确认。
不符合标准会自动重命名/转码后导入

拖拽视频后先做格式校验，再导入符合项。
提示不符合标准视频在部分设备可能播放异常。

快捷操作：
通过Alt + - 开启面板
支持以下功能：
动画延迟、快速命名、参数复制、原位粘贴
